# 📝 AI 영어 일기단어장

**모바일 프로그래밍 8조 팀 프로젝트**  
영어 일기를 쓰고, AI의 도움으로 더 나은 영어 학습을 경험하세요!

## 📱 주요 기능
- **일기 작성 및 관리 (CRUD)**  
  사용자가 영어 또는 한글로 일기를 작성하고 수정/삭제할 수 있습니다.

- **AI 문법 수정 기능**  
  잘못된 영어 문장을 AI가 자연스럽고 정확한 문장으로 수정해줍니다.

- **중요 단어/표현 추출 & 단어장 기능**  
  수정된 영어 일기에서 핵심 단어 및 표현을 추출해 단어장을 자동 생성합니다.
